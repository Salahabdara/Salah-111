حزمة كاملة v19 (Frontend + Backend API)
=======================================
النشر على استضافة PHP:
1) ارفع كل محتويات هذا الأرشيف داخل مجلد htdocs في الاستضافة.
2) أنشئ قاعدة بيانات MySQL واستورد ملف db/legacy_schema_v14.sql (إن وجد لديك سكربت أحدث فاستعمله).
3) حرّر ملف config.php داخل الجذر (إن وُجد) وأدخل بيانات قاعدة البيانات (host, dbname, user, pass).
4) واجهة الموقع في الجذر (index.html). الـ API موجود داخل مجلد /api.
5) ملف assets/config.js مضبوط على API_BASE_URL = /api (لا يلزم تعديل إذا الواجهة والخلفية على نفس الدومين).
6) إن ظهرت مشكلة أذونات، جرّب PHP 8.1 أو 8.2 من لوحة الاستضافة.

ملاحظات:
- مجلدا seeds/ و db/ يحتويان بيانات/سكربتات تجريبية.
- ملف api/templates/notice_no_access.html رسالة تحذير حمراء (صلاحيات التحميل).
- تستطيع حذف ملف padding.bin بعد النشر (ملف لرفع حجم الأرشيف فقط).
